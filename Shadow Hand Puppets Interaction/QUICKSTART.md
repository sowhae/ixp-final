```bash

# Start Claude Code

```

## Step 3: Run the Project

```bash
# Option A: Simple HTTP server
npm start

# Option B: Python (if Node.js not installed)
python3 -m http.server 8080

# Then open: http://localhost:8080
```

When prompted in your browser, click "Allow" for camera access.

## Step 5: Start Playing!
Move your hands and make gestures - you'll see them as shadows!
